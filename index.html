O que este projeto entrega

Análise exploratória

Notebook com 10 seções: da leitura e limpeza dos dados à interpretação e conclusão.

Dashboard interativo

App em Streamlit com filtros múltiplos, KPIs dinâmicos e gráficos interativos.

Persistência em banco

Dados armazenados em SQLite e acessados via SQLAlchemy.

Correlação estatística

Matriz de correlação entre variáveis numéricas com Pandas e Seaborn.

Funcionalidades técnicas

  • Intermediárias: filtros múltiplos, KPIs dinâmicos, gráficos interativos, análise temporal, seções organizadas.
  • Avançadas: persistência em banco (SQLAlchemy + SQLite) e correlação estatística (Pandas/NumPy).
  • Stack: Python, Pandas, Matplotlib, Seaborn, Plotly, Streamlit.

Como rodar

Clone o repositório e execute:

pip install -r requirements.txt  →  streamlit run app.py
